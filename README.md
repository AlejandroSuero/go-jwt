# Go-JWT [![JWT-logo]][jwt.io]

Decode and validate your [JWT](https://jwt.io) tokens in Go.

</div>

> [!WARNING]
>
> This is a work in progress. The CLI is not ready yet.

## Install

```bash
go install github.com/AlejandroSuero/go-jwt
```
